Create a new Domain

In order to use BubbleDNS for your own Domain, you need to set the glue record and nameservers to the following values:

  • DB-Load

  • DB-Load

It could take up to 24 hours until the changes get in action!

You can test it yourself with the windows command line:

  • Enter "nslookup"

  • Enter "set q=ns"

  • Enter "*domain*"

    • If both of the servers of BubbleDNS come up, you can now enter the domain on this site

    After our automated verification system confirms everything, the domain is ready to go
