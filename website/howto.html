Automatic Update

If you want to update the DNS-Entry according to your current IP-Adress, you can use the following Link:

https://[MainDomain]/update&id=[DNS-Entry-ID]&apikey=[API-Key]


If you want to update the DNS-Entry to a custom IP-Address, you can use the following Link:

https://[MainDomain]/update&id=[DNS-Entry-ID]&apikey=[API-Key]&value=[IP-Address]
